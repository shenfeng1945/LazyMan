LazyMan笔试题
```
LazyMan('Hank')输出：
Hi! This is Hank!

LazyMan('Hank').sleep(10).eat('dinner')输出：
Hi! This is Hank!
//等待10s..
Wake up after 10
Eat dinner~

LazyMan('Hank').eat('dinner').eat('supper')输出：
Hi! This is Hank!
Eat dinner~
Eat supper~

LazyMan('Hank').sleepFirst(5).eat('supper')输出：
//等待5s
Wake up after 5
Hi! This is Hank!
Eat supper~
```
